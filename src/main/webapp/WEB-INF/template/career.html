人才招聘

深圳经纬捷讯信息技术有限公司注册于前海自贸区，是一家基于互联网为客户提供软件服务的高新技术企业。

公司以客户为中心，依托云计算、大数据、车联网、物联网等信息技术，通过SaaS模式，为传统行业客户提供可靠的信息产品和全面的综合解决方案。

JAVA工程师

岗位职责
  1. 负责公司各类产品的系统分析、架构设计、功能开发及相关运营后台的建设维护
  2. IT支撑系统需求分析、产品设计、开发和维护
岗位要求
  1. 全日制大专及以上学历，计算机相关专业
  2. 3年以上Java开发经验，熟练使用设计模式
  3. 熟练使用Eclipse等开发工具，熟悉一种web服务器的安装部署，熟悉Linux的基本操作
  4. 熟悉Java语言和面向对象设计开发，熟悉多线程程序开发，对多线程技术、异步、并发有较深入理解；熟悉MySQL应用开发、配置、维护及性能优化

运维工程师

岗位职责
  1. 负责网络及其设备的维护、管理、故障排除等日常工作，确保公司网络日常的正常运作
  2. 负责公司办公环境的软硬件和桌面系统的日常维护
  3. 维护客户局域网、广域网，保证其正常运行，确保局域网、广域网在工作期间内安全稳定运行
  4. 安装和维护客户计算机、服务器系统软件和应用软件，同时为其他部门提供软硬件技术支持
  5. 解决排除各种软硬件故障，做好记录，定期制作系统运行报告；维护数据中心，对系统数据进行备份
岗位要求
  1. 全日制大专及以上学历，计算机相关专业

市场营销经理

岗位要求
  1. 负责分析、挖掘用户需求，设计产品功能、流程，制作产品原型，编写需求文档，协调整合资源，与协作部门保持良好沟通，及时跟踪开发进度，推进项目达成产品目标及质量。
  2. 跟踪产品运营数据，不断完善和优化产品，提升产品的用户体验
  3. 为人踏实细心，工作认真，具有敬业精神、沟通能力和团队合作精神
  4. 根据公司整体市场策略规划，拟定市场推广计划；熟悉电商平台各类传播渠道，有营销意识，对营销卖点有准确洞察；热爱文案创作，有相当的文字根基优先考虑
